# 🎯 StratTrust AI 快速体验指南

## 📱 访问方式

**项目地址**: http://localhost:3000

## 🗺️ 页面导航

### 1️⃣ 首页 (落地页)
**URL**: `/` 或 `http://localhost:3000`

**体验内容**:
- 📖 产品介绍和特色概览
- 🎯 核心功能四大模块展示
- 🔄 完整的工作流程说明
- 💡 竞争优势和技术栈介绍
- 📊 关键指标展示

**快速操作**:
- 点击 "立即体验" 按钮进入因子库
- 点击各个功能卡片直接跳转到对应模块
- 导航栏可快速访问各部分内容

---

### 2️⃣ 因子库 (📊 因子管理)
**URL**: `/factors` 或点击首页 "因子库管理" 卡片

**体验内容**:
- 📈 5 个预置因子（市值、情绪、流动性、持有者分布、交易量）
- 🏷️ 按标签筛选因子（市场、情绪、社交、流动性、DEX、基础面等）
- 📊 因子统计信息（总数、公开数、历史表现）
- 🎨 每个因子的详细卡片信息

**交互体验**:
1. 点击顶部 "+ 新增因子" 按钮创建新因子
2. 在弹出对话框中填写：
   - 因子名称
   - 描述
   - 标签（逗号分隔）
3. 点击各个标签按钮进行筛选
4. 点击因子卡片上的 "查看详情" 按钮

**Mock 数据示例**:
- **社交情绪因子**: v2.0.0, 82% 历史表现, 实时更新频率
- **流动性因子**: v1.5.0, 75% 历史表现, 每30分钟更新
- **市值因子**: v1.2.0, 78% 历史表现, 每小时更新

---

### 3️⃣ AI 推理引擎 (🤖 推理)
**URL**: `/infer` 或点击首页 "AI 推理引擎" 卡片

**体验内容**:
- ⚙️ 推理配置面板 - 选择要使用的因子
- 📊 关键性能指标卡片（预期回报、波动率、多样化指数、置信度）
- 📈 置信度趋势图表（使用 Recharts 图表库）
- 💡 5 个交易信号（BUY/SELL/HOLD）

**交互体验**:
1. 在 "推理配置" 部分勾选要使用的因子
2. 点击 "▶️ 运行推理" 按钮
3. 等待 1.5 秒（模拟推理延迟）后查看结果
4. 查看各个交易信号的详细信息
5. 点击 "✅ 确认策略执行" 或 "💾 保存为模板"

**推理结果示例**:
- **PEPE**: BUY, 87% 置信度, +35% 预期收益
- **SHIB**: BUY, 82% 置信度, +28% 预期收益  
- **DOGE**: HOLD, 71% 置信度, +8% 预期收益
- **FLOKI**: BUY, 79% 置信度, +32% 预期收益
- **BONK**: SELL, 75% 置信度, -15% 预期收益

**关键指标**:
- 预期回报: 20.2%
- 波动率: 18%
- 多样化指数: 78%
- 模型置信度: 82%

### 4️⃣ 交易策略 (⚙️ 策略构建)
**URL**: `/strategies` 或点击首页 "策略构建" 卡片

**体验内容**:
- 📋 3 个预置策略
- 📊 性能指标仪表板（总回报、YTD、Sharpe比率、最大回撤、胜率）
- 🏷️ 策略状态标签（运行中、测试中、已暂停）
- ➕ 创建新策略对话框

**交互体验**:
1. 点击 "+ 创建新策略" 按钮
2. 填写策略信息：
   - 策略名称
   - 描述
   - 选择的因子（逗号分隔）
3. 点击 "创建" 按钮
4. 点击已有策略的按钮：
   - "📊 查看详情" - 查看策略配置
   - "📈 回测" - 运行回测
   - "⚙️ 编辑" - 编辑策略参数

**预置策略示例**:
- **MEME币热度选币**
  - 状态: 运行中 ✅
  - 总回报: 35%, YTD: 24.5%, Sharpe: 1.82
  - 最大回撤: -18%, 胜率: 68%

- **价值投资策略**
  - 状态: 测试中 🧪
  - 总回报: 18%, YTD: 12.8%, Sharpe: 1.45
  - 最大回撤: -12%, 胜率: 62%

- **流动性挖矿策略**
  - 状态: 运行中 ✅
  - 总回报: 28%, YTD: 18.5%, Sharpe: 1.65
  - 最大回撤: -15%, 胜率: 71%

### 5️⃣ 审计追溯 (📋 审计)
**URL**: `/audit` 或点击首页 "审计追溯" 卡片

**体验内容**:
- 📊 审计统计信息（总操作、成功、待处理、成功率）
- 🔍 多维筛选（按操作类型、按状态）
- 📝 5 条完整的审计日志时间线
- 💾 导出功能（CSV、合规报告、链上存证）

**审计操作类型**:
- 🤖 **AI 推理**: 模型执行、因子版本、置信度
- ✅ **TEE 验证**: 签名、认证、时间戳
- ⚡ **交易执行**: 交易哈希、Gas使用、网络信息
- 💰 **结算**: 利润、分成、手续费

**交互体验**:
1. 使用筛选按钮过滤审计日志：
   - 按操作类型（推理、验证、执行、结算）
   - 按状态（全部、成功、待处理、失败）
2. 点击每条日志的 "▶️ 查看详情" 展开/收起详细信息
3. 查看操作哈希值用于链上验证
4. 点击导出按钮生成报告

**审计日志示例**:
- **推理日志**: trace_67890_abc123, 成功, 5个信号, 87% 置信度
- **验证日志**: TEE签名有效, ECDSA-256算法, Attestation有效
- **执行日志**: 交易成功, Tx: 0xabcd1234..., Gas: 125000
- **结算日志**: 利润0.0234, 分成0.0187, 手续费0.0047

## 🎮 交互演示流程

### 完整使用场景
1. **首页浏览** (1-2分钟)
   - 浏览产品介绍和功能概览
   - 查看工作流程说明

2. **因子库探索** (2-3分钟)
   - 浏览预置因子
   - 按标签筛选因子
   - 创建新的自定义因子

3. **推理体验** (3-5分钟)
   - 配置推理参数（选择因子）
   - 运行推理引擎
   - 查看交易信号和统计指标
   - 保存推理结果为模板

4. **策略管理** (2-3分钟)
   - 浏览已有策略的性能指标
   - 创建新策略
   - 查看策略详情和回测结果

5. **审计查看** (2-3分钟)
   - 浏览完整的操作日志
   - 按类型和状态筛选
   - 查看详细的操作信息
   - 导出审计报告

**总计体验时间**: 12-20 分钟
